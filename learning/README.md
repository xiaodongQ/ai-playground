## 📖 资料详解

### 📘 完全掌握指南 (openclaw-master-guide.md)

**8 大章节，覆盖全部知识点：**

1. **快速定位** - 找到你的学习起点
2. **核心架构** - 理解 OpenClaw 工作原理
3. **配置详解** - 掌握所有配置项
4. **工具系统** - 学会所有内置工具
5. **技能开发** - 创建自定义技能
6. **实战练习** - 动手实践
7. **提示词库** - 现成模板直接使用
8. **自我优化** - 持续改进

**适合场景：**
- 系统性学习
- 深入理解原理
- 查阅特定主题

---

### 🎯 提示词优化器 (prompt-optimizer.md)

**让 AI 帮你写出更好的提示词：**

- **原理篇** - 提示词工程基础
- **工具箱** - CRISP 框架、5W1H 检查法
- **模板库** - 5 大场景现成模板
- **高级技巧** - 思维链、少样本学习
- **实战练习** - 边学边练

**核心公式：**
```
优质提示词 = 角色 + 任务 + 约束 + 上下文 + 示例 + 输出格式
```

**适合场景：**
- 提升提示词质量
- 学习提示词工程
- 优化现有提示词

## 💡 学习建议

### ✅ 应该做的

- **边学边做** - 不要只看不练
- **记录问题** - 建立自己的 FAQ
- **定期复习** - 每周回顾所学
- **参与社区** - Discord 交流
- **分享输出** - 写博客/教别人

### ❌ 应该避免的

- **贪多嚼不烂** - 一天学太多
- **只看不练** - 不动手实践
- **遇到问题就放弃** - 坚持一下
- **不记录** - 学过就忘
- **闭门造车** - 不交流
