# 04-实践指南

> 安装部署、配置说明、使用示例和常见问题排查

## 4.1 安装部署

### 4.1.1 系统要求

**最低要求**:
- Python 3.11+
- 1GB RAM
- 100MB 磁盘空间
- Linux/macOS/Windows

**推荐配置**:
- Python 3.12
- 2GB RAM
- 系统守护进程 (systemd 或类似)

### 4.1.2 快速安装 (2 分钟)

#### 步骤 1: 克隆仓库

```bash
git clone https://github.com/ysz/nanoClaw.git
cd nanoClaw
```

#### 步骤 2: 创建虚拟环境

```bash
# Linux/macOS
python -m venv venv
source venv/bin/activate

# Windows
python -m venv venv
venv\Scripts\activate
```

#### 步骤 3: 安装依赖

```bash
# 基础安装
pip install -e .

# 开发模式 (包含测试依赖)
pip install -e ".[dev]"

# 带语义搜索 (可选)
pip install -e ".[semantic]"
```

#### 步骤 4: 验证安装

```bash
nanoclaw --help
```

**预期输出**:
```
Usage: nanoclaw [OPTIONS] COMMAND [ARGS]...

  nanoClaw - Ultra-lightweight secure AI assistant

Options:
  --help  Show this message and exit.

Commands:
  serve   Start the nanoClaw server
  init    Interactive setup wizard
  chat    Interactive chat (testing)
  status  Check nanoClaw status
  doctor  Run security audit
  cron    Manage scheduled tasks
```

### 4.1.4 手动配置

如果不想使用向导，可以手动编辑配置文件：

```bash
# 创建配置目录
mkdir -p ~/.nanoclaw

# 编辑配置文件
nano ~/.nanoclaw/config.json
```

**配置模板**:

```json
{
  "providers": {
    "openrouter": {
      "apiKey": "sk-or-YOUR_API_KEY"
    },
    "anthropic": {
      "apiKey": "sk-ant-YOUR_API_KEY"
    },
    "openai": {
      "apiKey": "sk-proj-YOUR_API_KEY"
    }
  },
  "agents": {
    "defaults": {
      "model": "anthropic/claude-sonnet-4"
    }
  },
  "channels": {
    "telegram": {
      "enabled": true,
      "token": "BOT_TOKEN_HERE",
      "allowFrom": ["YOUR_TELEGRAM_USER_ID"]
    }
  },
  "tools": {
    "webSearch": {
      "apiKey": "BRAVE_API_KEY"
    },
    "shell": {
      "enabled": true,
      "timeout": 30,
      "confirmDangerous": true
    }
  },
  "memory": {
    "maxHistory": 50,
    "semanticSearch": false
  },
  "agent": {
    "maxIterations": 15,
    "maxTokensPerSession": 50000,
    "sessionTimeout": 300
  },
  "dashboard": {
    "enabled": true,
    "port": 18790
  }
}
```

**设置文件权限**:

```bash
chmod 600 ~/.nanoclaw/config.json
```

#### 后台运行 (生产环境)

**使用 nohup**:

```bash
nohup nanoclaw serve > ~/.nanoclaw/nanoclaw.log 2>&1 &
```

**使用 systemd** (Linux):

```bash
# 创建服务文件
sudo nano /etc/systemd/system/nanoclaw.service
```

**服务内容**:
```ini
[Unit]
Description=nanoClaw AI Assistant
After=network.target

[Service]
Type=simple
User=your_username
WorkingDirectory=/home/your_username/nanoClaw
Environment="PATH=/home/your_username/nanoClaw/venv/bin"
ExecStart=/home/your_username/nanoClaw/venv/bin/nanoclaw serve
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

**启动服务**:
```bash
# 重载 systemd
sudo systemctl daemon-reload

# 启用服务
sudo systemctl enable nanoclaw

# 启动服务
sudo systemctl start nanoclaw

# 查看状态
sudo systemctl status nanoclaw

# 查看日志
sudo journalctl -u nanoclaw -f
```

**使用 Docker**:

```bash
docker run -d \
  --name nanoclaw \
  --restart unless-stopped \
  -v ~/.nanoclaw/config.json:/app/config/config.json:ro \
  -v ~/.nanoclaw/workspace:/app/workspace \
  -v ~/.nanoclaw/data:/app/data \
  -p 127.0.0.1:18790:18790 \
  nanoclaw/nanoclaw:latest
```

## 4.2 配置说明

### 4.2.1 完整配置项

```json
{
  // === LLM 提供商配置 ===
  "providers": {
    "openrouter": {
      "apiKey": "sk-or-xxx",
      "baseUrl": null  // 可选：自定义 API 端点
    },
    "anthropic": {
      "apiKey": "sk-ant-xxx"
    },
    "openai": {
      "apiKey": "sk-proj-xxx"
    },
    "deepseek": {
      "apiKey": "sk-ds-xxx"
    }
  },
  
  // === Agent 默认设置 ===
  "agents": {
    "defaults": {
      "model": "anthropic/claude-sonnet-4",
      "temperature": 0.7,  // 可选：温度参数
      "maxTokens": 4096    // 可选：最大输出 tokens
    }
  },
  
  // === 通信渠道 ===
  "channels": {
    "telegram": {
      "enabled": true,
      "token": "BOT_TOKEN",
      "allowFrom": ["USER_ID_1", "USER_ID_2"],
      "pollInterval": 1  // 轮询间隔 (秒)
    },
    "console": {
      "enabled": false  // 控制台测试模式
    }
  },
  
  // === 工具配置 ===
  "tools": {
    "webSearch": {
      "apiKey": "BRAVE_API_KEY",
      "maxResults": 5  // 最大搜索结果数
    },
    "shell": {
      "enabled": true,
      "timeout": 30,  // 超时 (秒)
      "confirmDangerous": true  // 危险操作需要确认
    },
    "files": {
      "workspaceOnly": true,  // 限制在工作区
      "maxFileSize": 1048576  // 最大文件大小 (1MB)
    }
  },
  
  // === 记忆配置 ===
  "memory": {
    "maxHistory": 50,  // 最大历史消息数
    "semanticSearch": false,  // 是否启用语义搜索 (需要 chromadb)
    "memoryLimit": 100  // 最大记忆数
  },
  
  // === Agent 行为 ===
  "agent": {
    "maxIterations": 15,  // 最大迭代次数
    "maxTokensPerSession": 50000,  // 每会话最大 tokens
    "sessionTimeout": 300,  // 会话超时 (秒)
    "cacheEnabled": true,  // 启用工具缓存
    "cacheTTL": 300  // 缓存 TTL (秒)
  },
  
  // === Dashboard ===
  "dashboard": {
    "enabled": true,
    "port": 18790,
    "host": "127.0.0.1"  // 仅 localhost
  },
  
  // === 日志 ===
  "logging": {
    "level": "INFO",  // DEBUG, INFO, WARNING, ERROR
    "file": "~/.nanoclaw/nanoclaw.log",
    "maxSize": 10485760,  // 10MB
    "backupCount": 3  // 保留 3 个备份
  }
}
```

### 4.2.3 模型选择

**推荐模型**:

| 使用场景 | 推荐模型 | 成本 | 速度 |
|---------|---------|------|------|
| 日常使用 | `anthropic/claude-sonnet-4` | 低 | 快 |
| 复杂任务 | `anthropic/claude-opus-4` | 高 | 中 |
| 代码生成 | `openai/gpt-5` | 中 | 快 |
| 性价比 | `deepseek/deepseek-chat` | 很低 | 快 |
| 本地运行 | `ollama/llama3.1` | 免费 | 中 |

**切换模型**:

```bash
# 方法 1: 修改配置文件
nano ~/.nanoclaw/config.json
# 修改 "model": "anthropic/claude-sonnet-4"

# 方法 2: 命令行覆盖
nanoclaw serve --model "openai/gpt-5"
```

## 4.4 常见问题排查

### 4.4.1 安装问题

**问题 1: Python 版本过低**
```
ERROR: Package 'nanoclaw' requires Python 3.11+
```

**解决**:
```bash
# 检查 Python 版本
python --version

# 升级 Python (Ubuntu/Debian)
sudo apt update
sudo apt install python3.11 python3.11-venv

# 使用 pyenv (推荐)
curl https://pyenv.run | bash
pyenv install 3.12.0
pyenv global 3.12.0
```

**问题 2: 依赖安装失败**
```
ERROR: Could not find a version that satisfies the requirement aiohttp>=3.9
```

**解决**:
```bash
# 升级 pip
pip install --upgrade pip

# 清理缓存
pip cache purge

# 重新安装
pip install -e . --no-cache-dir
```

**问题 3: 权限错误**
```
ERROR: Could not install packages due to an EnvironmentError: [Errno 13] Permission denied
```

**解决**:
```bash
# 不要使用 sudo! 使用虚拟环境
python -m venv venv
source venv/bin/activate
pip install -e .
```

### 4.4.2 配置问题

**问题 1: Telegram Bot 无法连接**
```
ERROR: Telegram connection failed: Unauthorized
```

**解决**:
```bash
# 1. 检查 Bot Token 是否正确
# 在 Telegram 中搜索 @BotFather, 发送 /mybots

# 2. 检查用户 ID 是否正确
# 在 Telegram 中搜索 @userinfobot, 发送任意消息获取 ID

# 3. 确保 Bot 已启动
# 在 Telegram 中搜索你的 Bot, 发送 /start
```

**问题 2: LLM API 连接失败**
```
ERROR: LLM API error 401: Invalid API key
```

**解决**:
```bash
# 1. 检查 API Key 是否正确
nano ~/.nanoclaw/config.json

# 2. 测试 API Key
curl -H "Authorization: Bearer sk-or-YOUR_KEY" \
     https://openrouter.ai/api/v1/auth/key

# 3. 检查余额
# 登录 provider 网站查看余额
```

**问题 3: 配置文件权限不安全**
```
WARNING: Config file permissions are insecure (644)
```

**解决**:
```bash
chmod 600 ~/.nanoclaw/config.json
```

## 4.6 小结

### 安装要点

1. **2 分钟安装**: 克隆 → 虚拟环境 → 安装 → 配置

3. **工具配置**: Web 搜索、Shell、文件操作
4. **性能调优**: 缓存、迭代次数、Token 预算

### 使用要点

1. **自然对话**: 像和人聊天一样使用
2. **工具调用**: 自动选择合适工具
3. **文件操作**: 限制在工作区
4. **后台任务**: 长任务异步执行
5. **定时任务**: Cron 风格调度

### 排查要点

1. **查看状态**: `nanoclaw status`
