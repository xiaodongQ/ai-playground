## 安全架构

```
┌─────────────────────────────────────────────────────────┐
│                    用户输入                              │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│  Layer 1: InputGuard (输入验证)                         │
│  - 长度限制                                               │
│  - 禁止命令模式                                           │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│  Layer 2: ShellSandbox (Shell 沙箱)                      │
│  - TIER 1: BLOCKED (永远阻止)                           │
│  - TIER 2: CONFIRM (需要确认)                           │
│  - TIER 3: EXECUTE (安全执行)                           │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│  Layer 3: FileGuard (文件守卫)                          │
│  - 工作区限制                                             │
│  - 敏感路径阻止                                           │
│  - 符号链接检查                                           │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│  Layer 4: PromptGuard (提示词注入防御)                   │
│  - NFKC 标准化                                           │
│  - 注入模式检测                                           │
│  - 输出清理包装                                           │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│  Layer 5: SessionBudget (会话预算)                       │
│  - Token 限制                                            │
│  - 速率限制                                               │
│  - 迭代次数限制                                           │
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────┐
│  Layer 6: AuditLog (审计日志)                           │
│  - 记录所有操作                                           │
│  - 异常检测                                               │
│  - 安全报告                                               │
└─────────────────────────────────────────────────────────┘
```

## ReAct 循环流程

```
用户消息
   │
   ▼
┌─────────────────┐
│ 加载上下文       │
│ - 历史 (15 条)    │
│ - 记忆 (5 条)     │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 动态选择工具     │
│ - 核心工具 (5 个)  │
│ - 条件工具       │
│ - 技能匹配       │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 调用 LLM         │
│ messages + tools│
└────────┬────────┘
         │
    ┌────┴────┐
    │         │
有工具调用   无工具调用
    │         │
    ▼         ▼
┌────────┐ ┌──────────┐
│并行执行│ │返回响应  │
│工具    │ │          │
└───┬────┘ └──────────┘
    │
    ▼
┌─────────────────┐
│ 添加工具结果     │
│ 到上下文         │
└────────┬────────┘
         │
         ▼
    迭代继续
    (最多 15 次)
```

## Token 优化策略

```
┌──────────────────────────────────────────────────────────┐
│  OpenClaw (优化前)                                        │
│                                                          │
│  System Prompt:    ~800 tokens                          │
│  Tool Schemas:    ~5000 tokens (100+ 工具)               │
│  History:        ~15000 tokens (50 条完整)               │
│  Tool Outputs:   ~8000 tokens (无截断)                   │
│  Memory Extract: 每条消息                               │
│                                                          │
│  Total:         ~28800 tokens / 对话                     │
│  Cost:          ~$0.05-0.10 / 对话                       │
└──────────────────────────────────────────────────────────┘

                    ↓ 优化

┌──────────────────────────────────────────────────────────┐
│  nanoClaw (优化后)                                        │
│                                                          │
│  System Prompt:    ~300 tokens (62% ↓)                  │
│  Tool Schemas:     ~800 tokens (84% ↓)                  │
│  History:         ~3000 tokens (80% ↓)                  │
│  Tool Outputs:    ~2000 tokens (75% ↓)                  │
│  Memory Extract: 30-40% 消息 (60-70% ↓)                 │
│                                                          │
│  Total:          ~6100-8100 tokens / 对话                │
│  Cost:           ~$0.01-0.02 / 对话                      │
│                                                          │
│  总节省：75-80%                                          │
└──────────────────────────────────────────────────────────┘
```
