### 6.3.3 配置优化

**Master 配置优化**：

```bash
# weed master 优化参数
weed master \
  -mdir=/data/master \
  -ip=10.0.3.1 \
  -port=9333 \
  -volumeSizeLimitMB=1024 \      # Volume 大小限制
  -garbageThreshold=0.3 \         # 垃圾回收阈值
  -maxParallelVacuumPerServer=4 \ # 并行 Vacuum 数
  -defaultReplication=001 \       # 默认 3 副本
  -raftHeartbeatInterval=300ms \  # Raft 心跳间隔
  -raftElectionTimeout=10s        # Raft 选举超时
```

**Volume Server 配置优化**：

```bash
# weed volume 优化参数
weed volume \
  -dir=/data/volume1 \
  -master=10.0.3.1:9333 \
  -port=9340 \
  -max=0 \                        # 自动创建 Volume
  -fixJpgOrientation=true \        # 自动修正图片方向
  -readMode=memory \              # 索引加载到内存
  -index=memory \                 # 内存索引
  -pulseSeconds=60                # 心跳间隔
```

**S3 Gateway 配置优化**：

```bash
# weed s3 优化参数
weed s3 \
  -filer=10.0.2.1:8888 \
  -ip=10.0.1.1 \
  -port=8333 \
  -allowedOrigins="*" \           # CORS 配置
  -domainName="s3.example.com" \  # 域名配置
  -localFiler=true \              # 本地 Filer 缓存
```

### 6.5.2 开发规范

**代码规范**：

```python
# 错误处理规范
try:
    await storage.upload_file(bucket, key, path)
except StorageError as e:
    logger.error(f"Upload failed: {e}")
    raise
except Exception as e:
    logger.exception(f"Unexpected error: {e}")
    raise StorageError(f"Upload failed: {e}")

# 重试规范
@retry(
    wait=wait_exponential(multiplier=1, min=1, max=10),
    stop=stop_after_attempt(3),
    retry=retry_if_exception_type(StorageTemporaryError)
)
async def upload_with_retry(storage, bucket, key, path):
    return await storage.upload_file(bucket, key, path)
```

**日志规范**：

```python
logger.error(f"Upload failed: {bucket}/{key}, error={error}, retry={retry_count}")
```
