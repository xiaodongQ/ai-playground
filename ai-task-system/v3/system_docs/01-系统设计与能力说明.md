# 系统设计与能力说明

## 一、系统能力概览

AI Task System V3 是一个专为个人开发者打造的 AI 任务自动化执行与管理平台。

### 核心能力

1. **任务管理**：创建、查看、删除任务
2. **自动领取**：新建任务自动执行
3. **定时轮询**：可选的定时轮询调度模式
4. **CodeBuddy 执行**：调用腾讯 CodeBuddy CLI 执行任务
5. **实时推送**：WebSocket 实时同步任务状态
6. **多轮对话**：支持任务续跑和迭代优化
7. **本地沉淀**：固定 Markdown 文件结构，永久留存

## 二、系统边界

### 支持的场景

- ✅ 单机内网环境使用
- ✅ 代码开发任务（Python、Go、Rust 等）
- ✅ 文档分析和总结任务
- ✅ 多轮对话迭代优化
- ✅ 批量任务定时执行

### 不支持的场景

- ❌ 多用户/多设备共享
- ❌ 外网云服务集成
- ❌ Docker/容器化部署
- ❌ 复杂的任务依赖图

## 三、模块职责

| 模块 | 职责 |
|------|------|
| `models.py` | 数据模型定义 |
| `storage.py` | SQLite 数据库封装 |
| `executor.py` | CodeBuddy CLI 执行器 |
| `scheduler.py` | 任务调度器 |
| `realtime.py` | WebSocket 实时推送 |
| `main.py` | FastAPI Web 服务 |
| `cli.py` | 命令行工具 |

## 四、CLI 命令规范

### 创建并执行任务

```bash
task-cli create -p "用 Python 写一个 FastAPI 接口"
```

### 续跑任务

```bash
task-cli resume a1b2c3d4 "优化代码结构"
```

### 任务管理

```bash
task-cli list
task-cli get a1b2c3d4
task-cli delete a1b2c3d4
```

### 调度器控制

```bash
task-cli scheduler start
task-cli scheduler stop
task-cli scheduler status
```
