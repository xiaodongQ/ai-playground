### 三、配置步骤

#### 3.1 Alpha Vantage（股票行情）

**步骤 1**: 注册获取 API Key
- 网址：https://www.alphavantage.co/support/#api-key
- 免费额度：25 次/天，5 次/分钟
- 等待时间：约 1-2 分钟收到邮件

**步骤 2**: 添加到 Gateway 配置

```bash
openclaw config.patch --note "添加 Alpha Vantage API Key"
```

配置内容：
```json
{
  "env": {
    "ALPHA_VANTAGE_API_KEY": "你的 API_KEY"
  }
}
```

**步骤 3**: 重启 Gateway

```bash
openclaw gateway restart
```

**步骤 4**: 更新 Fin·财多多 TOOLS.md

在 `/root/.openclaw/workspace-finance-agent/TOOLS.md` 中添加：

```markdown
## Alpha Vantage（股票行情）
- **API Key**: 存储在 Gateway 环境变量
- **文档**: https://www.alphavantage.co/documentation/
- **用途**: 股价、K 线、技术指标、财报数据
```

#### 3.2 CoinGecko（加密货币）

**无需配置，直接调用**：

```javascript
// 示例：获取比特币和以太坊价格
web_fetch("https://api.coingecko.com/api/v3/simple/price?ids=bitcoin,ethereum&vs_currencies=usd,cny")
```

**返回示例**：
```json
{
  "bitcoin": {"usd": 95234.5, "cny": 692345.2},
  "ethereum": {"usd": 3456.7, "cny": 25123.4}
}
```

---

#### 3.3 新浪财经（A 股行情）

**无需配置，直接抓取**：

```javascript
// 示例：获取贵州茅台（600519）行情
web_fetch("https://quote.eastmoney.com/sh600519.html")
```

**API 接口**（无需网页）：
```
http://hq.sinajs.cn/list=sh600519
```

**返回示例**：
```javascript
var hq_str_sh600519="贵州茅台，1680.00，1675.50，1680.00，1690.00，1670.00，...";
```

### 五、配额管理

| API | 免费额度 | 建议调用频率 | 优化策略 |
|-----|---------|-------------|---------|
| Alpha Vantage | 25 次/天 | ≤ 2 次/小时 | 缓存热点股票 |
| CoinGecko | 10-50 次/分 | 按需使用 | 无需优化 |
| Tavily | 已配置额度 | 按需使用 | 复用搜索结果 |

**缓存策略**：
- 热门股票价格：缓存 5 分钟
- 加密货币价格：缓存 1 分钟
- 财经新闻：缓存 30 分钟

---

### 六、错误处理

#### 常见问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| API 返回空数据 | 股票代码错误 | 验证股票代码格式 |
| 请求被限流 | 超过配额 | 等待后重试，增加缓存 |
| 网页抓取失败 | 反爬机制 | 更换数据源 |

#### 降级策略

```
Alpha Vantage（主） → 新浪财经（备用） → Tavily 搜索（兜底）
```
